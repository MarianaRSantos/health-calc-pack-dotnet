
HEALTH CALC
Esse projeto foi desenvolvido com a finalidade de calcular o IMC e os Macronutrientes de uma pessoa, diferenciando pelo Sexo. Como entrada,
é recebido o peso, a altura e o sexo. Como saída, é exibido a classificação do IMC, e as quantidades, em gramas, de proteínas, carboidratos e gorduras
baseado nos valores inseridos.
